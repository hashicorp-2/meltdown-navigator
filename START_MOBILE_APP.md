## Quick Start

### Option 1: Web Browser (Easiest)

```bash
cd apps/mobile
npm run web
```

Then open: **http://localhost:8081**

---

### Option 2: Expo Go (On Your Phone)

1. **Install Expo Go:**
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)
   - Android: [Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. **Start the app:**
   ```bash
   cd apps/mobile
   npm run start
   ```

3. **Scan QR Code:**
   - The terminal will show a QR code
   - Open Expo Go app
   - Tap "Scan QR Code"
   - Scan the code from terminal

---

### Option 3: iOS Simulator (Mac only)

```bash
cd apps/mobile
npm run ios
```

---

### Option 4: Android Emulator

```bash
cd apps/mobile
npm run android
```

---

## Configuration

The mobile app is configured to use:
- **Backend URL:** http://localhost:4000 (set in `.env`)

Make sure your backend is running before using the mobile app!
